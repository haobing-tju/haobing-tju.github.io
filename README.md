# Bing Hao Academic Homepage

This repository is prepared for `haobing-tju.github.io` using the Jekyll-style organization of `RayeRen/acad-homepage.github.io`.

## How to upload

1. Unzip this package locally.
2. Open the unzipped folder.
3. Upload the **contents inside the folder** to the root of your GitHub repository `haobing-tju/haobing-tju.github.io`.
4. Make sure the repository root directly contains `_config.yml`, `_pages/`, `_layouts/`, `_includes/`, `_data/`, `assets/`, `images/`, and `files/`.
5. Commit changes and wait for GitHub Pages to deploy.

## Main files to edit later

- `_pages/about.md`: homepage content, publications, awards, experience.
- `_config.yml`: title, author info, email, GitHub username, avatar path.
- `_data/navigation.yml`: top navigation links.
- `images/profile.jpeg`: profile photo.
- `files/Bing_Hao_CV.pdf`: CV PDF.

## Notes

- The page includes an English/Chinese toggle button.
- Publication links are placeholders (`#`) and should be replaced after arXiv/code links are available.
- Phone number, age, and political status are intentionally not included on the public homepage.
